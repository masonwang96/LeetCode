# 题目描述
给定一个正整数，检查他是否为交替位二进制数：换句话说，就是他的二进制数相邻的两个位数永不相等。

示例 1:
    
    输入: 5
    输出: True
    解释:
    5的二进制数是: 101
示例 2:

    输入: 7
    输出: False
    解释:
    7的二进制数是: 111
示例 3:

    输入: 11
    输出: False
    解释:
    11的二进制数是: 1011
 示例 4:

    输入: 10
    输出: True
    解释:
    10的二进制数是: 1010

### solution 1：
先转换为二进制，再判断二进制表示中是否含有‘00’、‘11’：
含有则返回False，不含有则返回True
